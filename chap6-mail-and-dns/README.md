Ubuntu版v1.0.0の「6章 メールサーバーの構築」を、Docker(docker-compose)で実現したものです。

メールサーバーの構築は、DNSサーバーの活用を前提としているため、先に「dns-with-docker」を先に構築してみると良いです。それが成功したら、Dockerコンテナを削除してから、本フォルダに取り組むと良いです。

本フォルダは、「chap5-dns」の内容をコピーして、メールサーバの構築分を追加したものです。

# Dockerコンテナ起動後

以下の手順で進めると良いです。

- host1でuser1を登録
- host2でuser2を登録
- user1とuser2間でメールを互いに送信して届くことを確認する
