教科書では、ホストOSからThunderbirdを用いてhost1に接続していましたが、今回はその代わりに、専用のコンテナからimapのCLIツールを用いてhost1に接続します。

# Dockerコンテナ起動後

以下の手順で進めると良いです。

- host1でuser1を登録（パスワードはuser1にすること）
- host2でuser2を登録（パスワードはなんでも良い）
- host2のuser2からhost1のuser1にメールを飛ばす。
- host1-imap-clientからmuttコマンドを入力して、メールをIMAPで見る（見ても消えない）。あるいはhost1からメールをPOP3で見る（見たら消える）。
